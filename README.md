# VoiceCanvas

VoiceCanvas gives a voice back to stroke survivors with aphasia. Users write freely on a guided canvas — Apple's Vision framework recognizes the handwriting and reads it aloud instantly via AVSpeechSynthesizer. Includes Quick Phrases for one-tap communication and full iPhone/iPad support. Built with SwiftUI, Vision, and AVFoundation. No internet required.

## Built With
- SwiftUI
- Apple Vision Framework  
- AVFoundation

## Requirements
- iOS 16.0+
- Swift Playgrounds 4+ or Xcode 15+
