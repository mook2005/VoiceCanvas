# VoiceCanvas
### *Voice from the Heart*

> An app playground that gives a voice back to those who have lost theirs.

VoiceCanvas was born from a personal experience caring for a grandfather with post-stroke aphasia — a condition that leaves the mind intact but silences the voice. This app transforms an iPad into an expressive communication tool, letting patients write naturally and be heard instantly.

---

## ✨ How It Works

**1. Write** — Draw freely on the large guided canvas using any finger. No pressure on neatness.

**2. Read** — Tap **Read Text** and Apple's Vision framework recognizes the handwriting on-device, instantly.

**3. Speak** — The recognized text is read aloud via AVSpeechSynthesizer — so caregivers can hear without needing to see the screen.

## ⚡ Features

- **Freehand Canvas** — Large guided writing area with smooth stroke rendering and adjustable brush sizes (S / M / L)
- **On-Device OCR** — Powered by Apple's Vision framework. No internet required. Patient data never leaves the device.
- **Instant Text-to-Speech** — Clear voice output via AVSpeechSynthesizer the moment recognition completes
- **Quick Phrases** — One-tap access to 12 common needs: Water, Toilet, Pain, Medicine, and more
- **Communication History** — Every spoken message is logged and can be replayed with a single tap
- **Fully Responsive** — Adapts fluidly across iPhone and iPad, portrait and landscape

## 📋 Requirements

- iOS 16.0+
- Swift Playgrounds 4+ **or** Xcode 15+
- No internet connection required

---

## 🚀 Getting Started

1. Clone this repository
2. Open `VoiceCanvas.swiftpm` in **Swift Playgrounds** (iPad or Mac) or **Xcode**
3. Run on a real device for full Vision and speech functionality

```bash
git clone https://github.com/mook2005/VoiceCanvas.git
```
